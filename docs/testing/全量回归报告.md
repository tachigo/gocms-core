| **期望** | user 对象应包含 `roles` 字段，如 `["admin"]` |
| **建议** | 在登录响应的 user 序列化中加入 roles 字段 |

### Bug-08: JWT Payload 缺少 roles 字段

| 项 | 说明 |
|-----|------|
| **严重级别** | 🟡 P1 |

| **期望** | payload 应包含 `roles` 字段，前端需要据此做权限判断 |
| **建议** | 在 JWT 签发时将用户角色列表写入 payload |

### Bug-11: 未认证请求返回 404 而非 401

| 项 | 说明 |
|-----|------|
| **严重级别** | 🟡 P1 |
